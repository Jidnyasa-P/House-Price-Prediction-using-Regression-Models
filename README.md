# House Price Prediction using Regression Models

A production-ready data science project that builds, trains, and deploys a regression model to predict property values based on key features.

## 🚀 Overview
This repository contains a full-stack implementation of a house price prediction system. It includes a Python-based machine learning pipeline for researchers and a live React + Node.js dashboard for end-users.

## 📊 Features
- **Predictive Engine**: Leverages Linear Regression and Random Forest algorithms.
- **Interactive Dashboard**: Real-time property valuation interface.
- **Data Visualization**: Correlation heatmaps, market trends, and feature importance.
- **API Access**: RESTful inference endpoint for integration.

## 🛠️ Tech Stack
- **Languages**: Python (Data Science), TypeScript (Frontend/API).
- **Libraries**: Scikit-Learn, Pandas, NumPy, Matplotlib, Seaborn.
- **Frameworks**: React, Express/FastAPI.
- **Visualization**: Recharts, D3.js.

## 📁 Project Structure
- `data/`: Sample datasets (Ames/Synthetic).
- `notebooks/`: Jupyter Notebooks for EDA and Model Tuning.
- `src/`: Dashboard source code.
- `models/`: Exported model weights/pickles.
- `api/`: Backend inference logic.

## 🧪 Implementation Workflow
1. **Data Cleaning**: Handling missing values and outliers.
2. **EDA**: Analyzing price distributions and feature correlations.
3. **Feature Engineering**: Encoding categoricals and scaling numericals.
4. **Model Selection**: Comparing Linear vs Random Forest vs XGBoost.
5. **Evaluation**: Metrics like MAE (Mean Absolute Error) and R² Score.

## 📈 Learning Outcomes
- Understanding multivariate regression analysis.
- Implementing feature selection techniques.
- Deploying ML models as web services.
- Building data-dense interactive dashboards.
